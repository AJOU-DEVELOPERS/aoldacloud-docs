# 공동 작업자 추가 가이드

## 공동 작업자 추가 방법

아올다클라우드에서 공동 작업자를 추가하려면, **문의하기** 버튼을 통해 설문 폼을 제출해야 합니다.  
아래 버튼을 클릭하면 설문 폼이 새 창에서 열립니다.

:::tip  
문의 폼 작성은 반드시 아주대학교 이메일(@ajou.ac.kr)을 사용해야 합니다.  
<Button className="custom-button" onClick={() => window.open('https://forms.gle/5ueUyGhUWiCM223r7', '_blank')}>문의하기 🤔</Button>
:::

---

## 설문 폼 작성 절차

### Step 1: 문의 유형 선택

설문 폼에서 가장 먼저 **문의 유형**을 선택해야 합니다.  
공동 작업자 추가를 요청하려면 **지원 요청**을 선택합니다.

---

### Step 2: 조직 이름 작성

:::note  
조직 이름이 기억나지 않을 경우, 클라우드 콘솔 페이지에서 확인할 수 있는 프로젝트 이름을 작성하세요.  
:::
문의 유형을 선택한 후, **조직 이름**을 작성합니다.  
조직 이름은 초기 프로젝트 생성 시 사용했던 이름과 동일하게 작성해야 합니다.

---

### Step 3: 문의 내용 작성
:::note  
문의는 프로젝트의 관리자 권한이 있는 사용자가 아니면 승인되지 않습니다.
:::
**문의 내용**을 작성합니다. 형식에 제한은 없지만, 요청 내용을 구체적이고 정돈되게 작성하면 신속한 처리가 가능합니다.  
아래는 공동 작업자 추가 요청에 대한 예시입니다:

<img src="/img/guide/intro/03/01-문의내용.png" alt="문의 내용 작성 예시" width="600"/>

---

### Step 4: 제출하기

모든 내용을 작성한 후, **제출하기** 버튼을 클릭하여 설문 폼을 제출합니다.  
문의는 지원 담당자가 확인 후 순차적으로 처리됩니다.

:::caution  
처리에는 하루에서 며칠이 소요될 수 있습니다. 여유를 가지고 요청해 주세요.  
:::

---

## 추가 참고사항

### 문제가 발생했을 경우

- 문의가 정상적으로 접수되지 않았다면, 입력한 이메일 주소와 내용을 다시 확인해 주세요.  
- 추가 문의 사항이 있거나 요청 진행 상황에 대한 확인이 필요하면 지원 담당자에게 직접 연락해 주세요.
