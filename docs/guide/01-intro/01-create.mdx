# 프로젝트 생성 가이드

아올다클라우드에서 **프로젝트**란 리소스(가상머신, 네트워크, 스토리지 등)를 할당받아 관리할 수 있는 **논리적인 작업 공간**입니다.  

아올다클라우드에서 사용자는 각자 자신의 프로젝트를 통해 클라우드 자원을 관리하고 사용할 수 있습니다. 이 가이드는 아올다클라우드에서 프로젝트를 생성하는 방법을 안내합니다.

## 프로젝트 생성 방법

### 1. 문의를 통한 프로젝트 생성

아올다클라우드에서는 프로젝트 생성을 위해 사용자가 **문의하기** 버튼을 통해 설문 폼을 제출해야 합니다.  
아래 버튼을 클릭하면 설문 폼이 새 창에서 열립니다.

:::tip  
문의 폼 제출은 아주대학교 이메일(@ajou.ac.kr) 계정으로만 가능합니다.  
<Button className="custom-button" onClick={() => window.open('https://forms.gle/5ueUyGhUWiCM223r7', '_blank')}>문의하기 🤔</Button>
:::

정상적으로 접속된 설문 폼은 아래와 같은 화면으로 나타납니다.

## 설문 폼 작성 절차

### Step 1: 문의 유형 선택

먼저, 설문 폼에서 **문의 유형**을 선택합니다.  
선택 가능한 유형은 다음과 같습니다:

1. 프로젝트 생성  
2. 리소스 쿼터 수정  
3. 지원 요청  
4. 오류 제보  
5. 기타  

**프로젝트 생성**을 선택하여 프로젝트 생성 요청을 진행합니다.

---

### Step 2: 조직 이름 작성

문의 유형을 선택한 후, **조직 이름**을 입력합니다.  
조직 이름은 프로젝트를 사용할 팀이나 단체가 명확히 식별될 수 있도록 작성해야 합니다.  

**조직 이름 작성 예시:**  
- 좋은 예: 중앙동아리 Do-iT  
- 나쁜 예: Do-iT  

:::note  
조직을 대표하는 관리자가 직접 문의를 작성해 주셔야 원활한 지원이 가능합니다.  
:::

---

### Step 3: 문의 내용 작성

마지막으로, **문의 내용**을 작성합니다.  
형식에 제한은 없지만, 구체적이고 정돈된 내용을 작성해 주시면 빠르게 처리할 수 있습니다.

아래는 프로젝트 생성에 대한 문의 내용 작성 예시입니다:

<img src="/img/guide/intro/01/02-문의내용.png" alt="문의 내용 작성 예시" width="600"/>

---

### Step 4: 제출하기

모든 내용을 작성한 후, **제출하기** 버튼을 눌러 설문 폼을 제출합니다.  
제출된 문의는 지원 담당자가 확인 후 순차적으로 처리됩니다.

:::caution  
요청 처리에는 하루에서 며칠이 소요될 수 있습니다. 여유를 가지고 요청해 주세요.  
:::

요청 진행 상황은 아래와 같이 실시간으로 확인됩니다.

## 추가 참고사항

### 프로젝트 생성 후 지원 절차
문의가 완료되면, 지원 담당자가 프로젝트 생성 및 관련 절차를 안내합니다.  
프로젝트 생성 외에도 리소스 추가 요청이나 오류 보고 등의 사항이 있으면 같은 절차로 문의하실 수 있습니다.


### 도움이 필요하세요?

가이드에 대한 질문이나 추가 요청 사항이 있다면, 언제든 문의해주세요!
