# 프로젝트 접속 가이드

아올다클라우드 프로젝트에 접속하려면 **사전에 프로젝트가 생성된 상태**여야 합니다.  
아직 프로젝트를 생성하지 않았다면, [프로젝트 생성하기](/guide/intro/create)를 참고하여 프로젝트를 생성한 후 접속 절차를 진행하세요.

## 클라우드 콘솔 접속 방법

### Step 1: 클라우드 콘솔 페이지 접속

클라우드 프로젝트에 로그인하고 리소스를 사용하려면 **클라우드 콘솔 페이지**에 접속해야 합니다.  
우측 상단의 **[콘솔 이동하기]** 버튼을 클릭하여 콘솔 페이지로 이동합니다.

:::info  
아래 버튼을 클릭하면 바로 아올다클라우드 콘솔 페이지로 이동할 수 있습니다.  
<Button className="custom-button" onClick={() => window.open('https://console.aoldacloud.com', '_blank')}>콘솔 이동하기</Button>
:::

---

### Step 2: 클라우드 콘솔 로그인

아올다클라우드 콘솔 로그인 페이지는 아래와 같이 표시됩니다.  
지원 담당자로부터 받은 계정 정보를 사용하여 로그인합니다.  

<img src="/img/guide/intro/02/01-콘솔로그인.png" alt="콘솔 로그인 페이지" width="600"/>

---

### Step 3: 클라우드 콘솔 확인

정상적으로 로그인하면 아래와 같은 콘솔 메인 페이지가 표시됩니다.  
여기에서 리소스 상태와 쿼터 설정이 올바른지 확인하세요.  

<img src="/img/guide/intro/02/02-콘솔메인페이지.png" alt="콘솔 메인 페이지" width="800"/>

---

## 추가 참고사항

### 프로젝트 접속 중 문제가 발생한 경우

- 로그인 정보가 올바르지 않다면 지원 담당자에게 문의하세요.  
- 콘솔 화면이 정상적으로 표시되지 않는 경우, 네트워크 상태를 확인하거나 재시도해보세요.
